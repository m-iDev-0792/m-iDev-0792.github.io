Projects

  HJRT HJRT is a portable C++ path tracing renderer which supports multi-material, multi-thread, BVH, depth of field, obj model loading and participate media. HJRT uses Dear Imgui as program GUI to show real-time rendering result and progress.
  PlantCreator PlantCreator is an implementation of my bachelor thesis -- Sketch-based fast plant modeling. User can create 3D plant models with a few sketch strokes by using PlantCreator.
  HJGraphics HJGraphics is a very simple 3D graphic framework based on OpenGL API. By using HJGraphics, you can create OpenGL environment and draw some funny 3D objects with a few lines of code.
  HJ Compiler HJ language is a C-like language. This program includes a compiler for HJ language, an IDE with syntax highlighting and code completion and a virtual machine to run HJ codes.
  HJ Editor HJ editor is a simple C++ editor with syntax highlighting and code completion;
  Gomoku A Gomoku AI implemented with game tree, Alpha-Beta pruning, Heuristic searching, Zorbrist Hash and multi-thread techniques.
  Ray Tracing Renderer A very basic ray tracing renderer that rendering Cornell Box scene. Monte-Carlo method, antialias, multi-thread and filter were used in this program.
