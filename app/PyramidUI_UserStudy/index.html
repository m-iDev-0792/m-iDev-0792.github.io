User Study Checklist

Complete the checklist below in order. You can tick each checkbox as you finish a step so the full process stays organized.

Before You Start

The study includes five tasks. You may see short practice rounds before the formal rounds, and the Next button stays disabled until every required response is completed. After the final task, keep the program open while it uploads your study data automatically.

Study Contact

If you have questions, need help finding your data file, or are unsure which materials to send, contact zbhe96@student.ubc.ca.

Confirm That the Study Data Was Uploaded Successfully

After the last task, keep the program open until the upload page clearly says it is safe to close the application.

Step 4
If the automatic upload succeeds

The upload page will report completion, and the local study data has already been sent to the server.

If the automatic upload fails

The User Study Application will help you locate the local .db file that contains your experiment data. The file path is copied to the clipboard automatically, and the upload page enables an Open Data File button so you can reveal the file in the system file manager.

  • Look for a file named like study_YYYYMMDD_HHMMSS.db.
  • Keep that .db file so it can be sent to the study organizer if requested.
  • Do not close the app until the organizer confirms the next step.
Wait for upload confirmation before closing
